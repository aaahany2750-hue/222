# AI-Powered Web Application (React + Node + MongoDB)

Complete full-stack starter featuring:
- Futuristic responsive UI with Tailwind CSS
- Login/signup with JWT auth
- User dashboard
- AI chatbot endpoint integration layer
- Voice input (Web Speech API)
- Dark mode settings
- Admin panel
- Notifications + settings storage

## Tech Stack
- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express, MongoDB (Mongoose)

## Run locally
```bash
npm install
npm install --prefix server
npm install --prefix client
cp server/.env.example server/.env
npm run dev
```

Frontend: `http://localhost:5173`  
Backend: `http://localhost:5000`
